# Ray - Mixer Comment Viewer
A comment viewer for the lagless streaming service [Mixer](https://mixer.com/).

## Dependencies
NW.js v0.19.5

Required modules:
- beam-client-node
- madaddress
- crypto

## i18n
Modify files under i18n directory or add files to add languages.
