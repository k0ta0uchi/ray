# Ray - Beam Comment Viewer
A comment viewer for the streaming service Beam.pro.

## Dependencies
NW.js v0.19.5

Required modules:
- beam-client-node
- madaddress
- crypto

## i18n
Modify files under i18n directory or add files to add languages.
